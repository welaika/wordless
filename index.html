Wordless is an opinionated Wordpress plugin that dramatically speeds up your custom theme creation.

Features

  • A structured and clean theme organization
  • Haml views (with support to layouts and partials)
  • Sass & Compass stylesheets
  • Coffeescript-powered logic (optional)
  • 50+ Rails-like helper functions
  • Released under the MIT license

Requirements

  • Wordpress .htaccess file present
  • Ruby (any version, full RVM support)
  • Compass and Sprockets gems installed
