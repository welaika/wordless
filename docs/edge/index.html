Introduction

Welcome to the developer documentation for Wordless. Actually you are looking at a complex WordPress plugin which make you create themes using the power of HAML, SASS/SCSS, COMPASS, and CoffeScript, drastically speeding up your development time!

Automation is for robots, we are humans, after all! ;)

Please refer to the online documentation for more details and instructions.
